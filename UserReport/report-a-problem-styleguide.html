1 — Color Palette

All colors meet WCAG AA (4.5:1) minimum; primary text pairs meet AAA (7:1+)

Brand Yellow #FCE11B Notes textarea background
Yellow Dark #c9b100 Notes textarea border, focus border
Yellow Text #3a2e00 Text on yellow background (10.2:1 contrast)
White #ffffff Card background, default tile background, Cancel button
Off-White #f4f5f7 Page background, footer bar, device strip
Text Primary #1a1f27 Default tile label, body text (15.3:1 on white)
Text Secondary #4a5260 Cancel button text, notes label, subtext
Text Muted #717b8c Device IP strip, placeholder hints
Border Default #c8cdd4 Unselected tile border, divider lines
Industrial UX note: No pure black (#000) or pure white (#fff) are used for backgrounds or large text surfaces. Research shows that in bright warehouse lighting, diffuse glare reduces perceived contrast — dark gray text on white (#1a1f27 on #ffffff) outperforms black-on-white in real-world readability tests. Red has been intentionally excluded as a UI color because under bright fluorescent and LED warehouse lighting, red reads as near-black at a glance.

Viewport meta tag required: Without this, mobile browsers scale the page down and all sizing breaks.
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">

min-height: Use 100dvh (dynamic viewport height) not 100vh — on mobile, 100vh includes the browser chrome bar and causes layout overflow. dvh accounts for the actual visible area.

6 — Touch Target Guidelines

Designed for gloved hands in warehouse environments
ElementMin HeightThis UIStandard
Issue Tiles64pxclamp(64px, 14vw, 88px)Apple HIG: 44px · Google: 48dp
Cancel Button52pxclamp(14px, 3.5vw, 18px) padding × 2 + fontExceeds minimums
Submit Button52pxSame padding, 2× width of CancelExceeds minimums
Textarea80pxclamp(80px, 18vw, 110px)N/A
Glove accommodation: Tiles are set to a 2-column grid with generous gap spacing. The minimum 64px tile height ensures reliable activation even with standard work gloves. Submit is 2× the width of Cancel to prevent accidental cancellation. The -webkit-tap-highlight-color: transparent and transform: scale(0.96) on :active provide tactile feedback without the default mobile blue flash.

One-handed use: Submit and Cancel are anchored in a sticky footer at the bottom of the screen — within natural thumb reach on all common device sizes. Critical actions should never require reaching to the top of the screen.

7 — Submit Payload & API Integration

The form serializes to JSON on submit. Wire the fetch call to your endpoint by uncommenting and updating the URL in the script block.

FieldTypeExampleNotes
issuesstring[]["Slower", "Connection"]Array of selected tile values. Empty array if none.
notesstring"Scanner keeps dropping"Empty string if textarea left blank.
ipstring"10.0.1.42"Fetched from api.ipify.org on load. Replace with internal IP resolution if needed.
timestampstring (ISO 8601)"2026-04-27T14:32:00.000Z"UTC timestamp generated client-side.
To activate: In the <script> block, find the commented-out fetch() call and replace /api/report with your endpoint URL. The payload object is already constructed above it.

Validation: Client-side check requires at least one tile selected OR a non-empty notes field before submit. A red hint message appears if neither condition is met. Server-side validation should mirror this.

8 — Accessibility & Industrial UX Principles

RequirementImplementation
WCAG contrast (AA)All text pairs exceed 4.5:1. Primary text (#1a1f27 on #fff) is 15.3:1.
WCAG contrast (AAA target)All primary/secondary text pairs exceed 7:1.
No color-only signalingSelected tiles use both color change AND border width change (2.5px → 3px)
No gradientsZero gradient usage — flat fills only, per industrial UX glare guidelines
No shadows on interactive elementsOnly card has a box-shadow (decorative). Tiles and buttons are flat.
Touch target sizeAll targets ≥ 64px height, exceeding Apple (44px) and Google (48dp) minimums
Font scalingAll font sizes use clamp() with rem-compatible minimums
No red as a signal colorRed is excluded. Under bright warehouse lighting, red reads as dark gray.
Prevent zoom on input focusmaximum-scale=1.0 in viewport meta prevents iOS zoom on textarea tap
Overscrolloverscroll-behavior: none prevents pull-to-refresh accidents
